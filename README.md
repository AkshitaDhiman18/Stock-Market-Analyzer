# 📈 Stock Market Analyzer

## 🚀 Real-World Stock Market Analysis Project using Python, NumPy & OOPs

A professional stock market analysis system built using **Python**, **NumPy**, **OOPs**, and **Matplotlib** to perform:

- 📊 Market Trend Analysis
- 📈 Investment Growth Simulation
- ⚠️ Risk & Volatility Analysis
- 💹 Profit/Loss Tracking
- 🧠 Buy/Sell/Hold Recommendation System
- 📉 Data Visualization

This project demonstrates practical implementation of **NumPy operations**, **data analysis concepts**, and **object-oriented programming** in a real-world finance-based application.

# 🔥 Project Features

## ✅ Stock Market Analysis
- Highest Closing Price Detection
- Lowest Closing Price Detection
- Average Price Analysis
- Daily Profit/Loss Calculation
- Percentage Change Analysis

## 📈 Trend Analysis
- Uptrend Detection
